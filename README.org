* Intro :noexport:
This file is written in [[https://orgmode.org/][Org Mode]] markup and may look funny in GitHub. But actually, this file alone is the *blogging engine* for https://ja.nz. It features:
- callable source blocks for templating
- clean up and helper functions
- markdown export with [[https://github.com/kaushalmodi/ox-hugo][Ox-Hugo]]

Org mode is really powerful combined with [[https://orgmode.org/manual/Working-with-Source-Code.html][source block]] snippets. Open this README with emacs and have a look yourself. I may blog about this one day too.

Sidenote: I am not using [[https://gohugo.io/][Hugo]] for blogging anymore. Too convoluted for my taste. So I wrote a barebone *markdown2blog* engine myself [[https://www.npmjs.com/package/gh-cms][GH-CMS]]

** New micro
#+NAME: micro
#+BEGIN_SRC emacs-lisp :var id=(read-string "id: ")
  (setq id (format "%s-%d-m" (downcase id) (length (org-map-entries t "LEVEL=2" nil))))
  (org-insert-heading)
  (insert id)
  (org-metaright)
  (org-todo "DRAFT")
  (org-set-property "export_file_name" id)
  (next-line)
  (insert "\n")
#+END_SRC

#+NAME: post
#+BEGIN_SRC emacs-lisp :var url=baseurl :var id=(read-string "id: ")
  (setq id (format "%s-%d-p" (downcase id) (length (org-map-entries t "LEVEL=2" nil))))
  (org-insert-heading)
  (insert id)
  (org-metaright)
  (org-todo "TODO")
  (org-insert-structure-template "description")
  (org-set-property "export_file_name" id)
  (next-line)
  (insert "\n")
  ;; Begin meta table
  (org-insert-heading)
  (insert "meta :noexport:")
  (org-metaright)
  (insert "\n")
  (org-table-create "2x2")
  (org-cycle)
  (insert "Asset")
  (org-cycle)
  (insert "Description")
  (org-cycle)
  (insert (format "%s/%s" (s-trim url) id))
  (org-cycle)
  (insert "Location")
  (org-cycle)
  (insert "file:static/")
  (org-cycle)
  (insert "Ref to assets")
  (org-cycle)
  (previous-line 6)
  ;; End meta table
#+END_SRC
