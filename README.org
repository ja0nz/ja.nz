* Intro :noexport:
This file is written in [[https://orgmode.org/][Org Mode]] markup and may look funny in GitHub. But actually, this file alone is the *blogging engine* for https://ja.nz. It features:
- callable source blocks for templating
- clean up and helper functions
- markdown export with [[https://github.com/kaushalmodi/ox-hugo][Ox-Hugo]]

Org mode is really powerful combined with [[https://orgmode.org/manual/Working-with-Source-Code.html][source block]] snippets. Open this README with emacs and have a look yourself. I may blog about this one day too.

Sidenote: I am not using [[https://gohugo.io/][Hugo]] for blogging anymore. Too convoluted for my taste. So I wrote a barebone *markdown2blog* engine myself [[https://www.npmjs.com/package/gh-cms][GH-CMS]]

** New tweet
#+NAME: tweet
#+BEGIN_SRC emacs-lisp :var ts=(format-time-string "%Y%m%d%H%M")
  (org-insert-heading)
  (org-metaright)
  (org-todo "DRAFT")
  (org-id-get-create)
  (org-set-property "CATEGORY+" ts)
  (org-set-property "export_file_name" ts)
#+END_SRC

** New post
#+NAME: post
#+BEGIN_SRC emacs-lisp
  (org-insert-heading)
  (org-metaright)
  (org-todo "TODO")
  (org-insert-structure-template "description")
  (org-id-get-create)
  (org-set-property "CATEGORY+" "SET")
  (org-set-property "export_file_name" "SET")
  (org-edit-headline "Set props!")
#+END_SRC

#+BEGIN_SRC emacs-lisp :var url=staticfar :results none
  (org-toggle-link-display)
  (replace-regexp "file:static" (s-trim url))
  (pop-global-mark)
#+END_SRC

*** Code Blocks and other blocks

I love to explain via src blocks
#+attr_html: :class my-section :id section-a
#+begin_src javascript
// class="highlight"
const answer = 42;
console.log("this is how we do")
#+end_src

Sometimes things express well with a wise (wo)mans quote:
#+begin_quote
It shall be light - and it was light mode and \<blockquote\>
#+end_quote

#+begin_verse

#+end_verse

#+begin_example
Let me show to you.... class="highlight"
#+end_example
