A nonempty string Record. Guarantees to return a string or empty/neutral string

```typescript
const safeRecord = new Proxy(
  {}, {
    get: (handler: Record<PropertyKey, string>, name: PropertyKey) : string => {
      handler[name] ??= ""; return handler[name];
    }
  }
);
```
